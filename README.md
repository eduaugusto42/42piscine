# 42 Piscine

This repository contains my work during the **42 Piscine**, an intensive coding bootcamp focused on learning the fundamentals of programming in C, shell scripting, and computer science basics.

## 📚 About the Piscine

The 42 Piscine is a 26-day selection and training program where participants solve progressively harder programming exercises under strict constraints:

- No external libraries (standard C library only when allowed)
- Strict coding norms (42 Norminette)
- Emphasis on logic, problem solving, and self-learning
- Peer evaluation and daily exercises
